## How to Use This Portfolio

### GitHub Pages (Recommended)
1. Fork or clone this repo to your GitHub account
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch** → `main` → `/ (root)`
4. Your portfolio will be live at `https://[username].github.io/[repo-name]`

### Adding Leonardo AI Images
Once you generate and convert images to `.webp`, drop them into `assets/images/`:
- `assets/images/hero-banner.webp` — hero section background
- `assets/images/profile.webp` — profile photo (optional)
- Then update `index.html`: replace the `.hero-img-placeholder` div with `<img src="assets/images/hero-banner.webp" ...>`
